## 4. 排版（Typography）

推荐字体族（Web/跨平台）:

```
--font-family: -apple-system, BlinkMacSystemFont, "SF Pro Text", "Helvetica Neue", Arial, "PingFang SC", "Microsoft YaHei", sans-serif;
```

字号与行高（常用）:

| 语义 | 字号 | 行高 |
|-----:|-----:|-----:|
| Display / H1 | 34px | 41px |
| H2 | 28px | 34px |
| H3 | 22px | 28px |
| Body Large | 18px | 27px |
| Body / Base | 16-17px | 24px |
| Small / Caption | 12-14px | 16-21px |

使用原则:
- 不超过 5 个字号层级
- 正文行高 >= 1.5 倍字号
- 行长控制：中文 30-50 字/行

---

## 5. 间距（Spacing）

采用 8px 网格，常用变量：4/8/12/16/24/32/48/64 px（示例变量见项目 CSS 变量）。

规则:
- 所有间距尽量使用 8 的倍数（4px 仅用于超紧凑场景）
- 组件内间距常用 8/12/16px
- 模块间距常用 32/48px

---

## 6. 圆角（Radius）与阴影（Shadow）

圆角常用值:
- small: 4-8px (按钮、输入)
- medium: 12px (卡片、模态)
- large: 16-20px (大卡片)
- full: 9999px (头像)

阴影常用范例（要求轻柔、自然）:
- sm: 0 1px 2px rgba(0,0,0,0.05)
- md: 0 4px 6px rgba(0,0,0,0.10)
- lg: 0 10px 15px rgba(0,0,0,0.12)

使用原则: 阴影用于层次表达，不用于装饰；尽可能使用简单的单层阴影以降低渲染成本。

---

## 7. 交互动效（Timing & Easing）

常用持续时间:
- instant: 100ms（按钮点击）
- fast: 150ms（颜色/图标变化）
- normal: 200ms（模态/页面过渡）
- slow: 300ms（复杂动画）

常用缓动:
- ease-standard: cubic-bezier(0.4, 0.0, 0.2, 1)
- ease-in-quick: cubic-bezier(0.4, 0.0, 1, 1)
- ease-out-slow: cubic-bezier(0.0, 0.0, 0.2, 1)

原则:
- 动效有明确目的（反馈/状态/空间关系）
- 动效应可中断且不阻塞用户操作

## 8. 组件命名、变体与导出约定

- 所有组件在 Figma 中应使用 `PascalCase` 命名，例如 `Button/Primary`、`Input/Text`。
- 每个组件至少包含三种状态（default、hover、disabled）与至少三种尺寸（sm、md、lg）视情况而定。
- 导出资源时遵循语义化命名（例如 `btn-primary@2x.png`、`icon-search.svg`）。
