## 🎨 颜色系统

### 1. 主色板 (Primary Palette)

遵循 Apple Design System，采用**语义化颜色**，支持浅色和深色模式。

#### 1.1 主品牌色 (Brand Colors)

| 名称 | 浅色模式 | 深色模式 | 用途 | 使用场景 |
|------|---------|---------|------|----------|
| **Primary Blue** | `#007AFF` | `#0A84FF` | 主要操作 | 按钮、链接、激活状态 |
| **Secondary Gray** | `#8E8E93` | `#A1A1A6` | 次要文本 | 辅助说明、禁用状态 |
| **Background** | `#FFFFFF` | `#000000` | 背景色 | 页面背景 |
| **Card Background** | `#F2F2F7` | `#1C1C1E` | 卡片背景 | 弹窗、列表项 |

#### 1.2 功能色 (Functional Colors)

| 名称 | 浅色 | 深色 | 语义 | 示例 |
|------|------|------|------|------|
| **Success** | `#34C759` | `#32AE47` | 成功 | ✅ 完成提示 |
| **Warning** | `#FF9500` | `#FF9500` | 警告 | ⚠️ 需注意 |
| **Error** | `#FF3B30` | `#FF453A` | 错误 | ❌ 操作失败 |
| **Info** | `#00C7BE` | `#30B0C0` | 信息 | ℹ️ 说明信息 |

#### 1.3 文本色 (Text Colors)

| 名称 | 浅色 | 深色 | 对比度 | 用途 |
|------|------|------|--------|------|
| **Primary Text** | `#000000` | `#FFFFFF` | ≥7:1 | 标题、主要文本 |
| **Secondary Text** | `#3C3C43` | `#EBEBF0` | ≥4.5:1 | 次要文本、描述 |
| **Tertiary Text** | `#8E8E93` | `#A1A1A6` | ≥3:1 | 禁用状态、占位符 |

#### 1.4 中性色阶 (Neutral Grays)

| 级别 | 浅色 | 深色 | 用途 |
|------|------|------|------|
| **Gray 1** | `#F2F2F7` | `#1C1C1E` | 轻微分隔 |
| **Gray 2** | `#E5E5EA` | `#2C2C2E` | 分割线、边框 |
| **Gray 3** | `#D1D1D6` | `#3A3A3C` | 禁用背景 |
| **Gray 4** | `#C7C7CC` | `#48484A` | 图标禁用 |
| **Gray 5** | `#ACACB1` | `#636366` | 时间戳 |

### 2. 颜色应用规则

#### 2.1 模式切换

```css
/* 浅色模式 (默认) */
:root {
  --color-primary: #007AFF;
  --color-bg: #FFFFFF;
  --color-text: #000000;
}

/* 深色模式 */
@media (prefers-color-scheme: dark) {
  :root {
    --color-primary: #0A84FF;
    --color-bg: #000000;
    --color-text: #FFFFFF;
  }
}
```

#### 2.2 无障碍访问 (WCAG AA)

- ✅ **文本对比度**: 大文本 3:1，普通文本 4.5:1，UI组件 3:1
- ✅ **颜色独立性**: 不仅用颜色传达信息，配合图标/文本
- ✅ **焦点状态**: 所有可交互元素需要清晰的焦点框

#### 2.3 禁用状态

```
禁用背景: Gray 3 + 40% 透明度
禁用文本: Tertiary Text + 50% 透明度
禁用图标: Gray 4
```

## 🔤 排版规范

### 1. 字体系统

#### 1.1 主要字体族

| 用途 | macOS | iOS | Android | Web备选 |
|------|-------|-----|---------|---------|
| **系统默认** | SF Pro Display | SF Pro Display | -apple-system | -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto |
| **等宽** | SF Mono | SF Mono | Roboto Mono | "SFMono-Regular", Consolas, "Courier New" |

#### 1.2 字体加载策略

```css
/* 本地系统字体 (无需加载) */
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, 'Helvetica Neue', Arial, sans-serif;

/* 等宽字体 (代码展示) */
font-family: 'SFMono-Regular', Consolas, 'Courier New', monospace;
```

### 2. 排版尺度 (Type Scale)

遵循 **8pt 基础网格**和 **1.25倍增长率**。

#### 2.1 标题级 (Heading)

| 级别 | 字号 | 行高 | 字重 | 字距 | 使用场景 | 示例 |
|------|------|------|------|------|---------|------|
| **H1** | 34px | 41px | 700 | -0.02em | 页面主标题 | 爱问首页标题 |
| **H2** | 28px | 34px | 700 | -0.02em | 模块标题 | 资源分类标题 |
| **H3** | 22px | 28px | 600 | -0.01em | 子标题 | 话题详情标题 |
| **H4** | 18px | 24px | 600 | 0em | 小标题 | 卡片标题 |

#### 2.2 正文级 (Body)

| 级别 | 字号 | 行高 | 字重 | 用途 | 示例 |
|------|------|------|------|------|------|
| **Body Large** | 18px | 24px | 400 | 主要内容 | 弹窗文本 |
| **Body Regular** | 16px | 24px | 400 | 标准文本 | 列表项文本 |
| **Body Small** | 14px | 20px | 400 | 次要文本 | 时间戳、说明文字 |

#### 2.3 标签级 (Label)

| 级别 | 字号 | 行高 | 字重 | 用途 | 示例 |
|------|------|------|------|------|------|
| **Label Large** | 12px | 16px | 500 | 按钮文本 | 操作按钮 |
| **Label Medium** | 11px | 16px | 500 | 辅助标签 | 标签、徽章 |
| **Label Small** | 10px | 14px | 500 | 特殊标签 | 角标、计数 |

### 3. 排版应用规则

#### 3.1 CSS实现

```css
/* H1 - 页面标题 */
.heading-1 {
  font-size: 34px;
  font-weight: 700;
  line-height: 1.2;
  letter-spacing: -0.02em;
}

/* Body Regular - 标准文本 */
.body-regular {
  font-size: 16px;
  font-weight: 400;
  line-height: 1.5;
  letter-spacing: 0;
}

/* Label Large - 按钮文本 */
.label-large {
  font-size: 12px;
  font-weight: 500;
  line-height: 1.33;
  letter-spacing: 0;
}
```

#### 3.2 行距规则

```
行高 = 字号 × 倍率
- 标题: 1.2倍 (较紧)
- 正文: 1.5倍 (适度)
- 标签: 1.33倍 (紧凑)
```

#### 3.3 文本缩放限制

```
最小字号: 10px (标签级)
最大字号: 34px (H1标题)
不使用自定义缩放 (使用rem相对单位)
```

#### 3.4 多行文本处理

```css
/* 标题截断 (单行) */
.text-truncate {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}

/* 多行截断 (2行) */
.text-clamp-2 {
  display: -webkit-box;
  -webkit-line-clamp: 2;
  -webkit-box-orient: vertical;
  overflow: hidden;
}
```

### 4. 排版响应式设计

#### 4.1 断点设置

| 设备 | 宽度 | 字号调整 |
|------|------|---------|
| **Mobile** | < 640px | H1: 28px, Body: 14px |
| **Tablet** | 640px - 1024px | H1: 32px, Body: 16px |
| **Desktop** | > 1024px | H1: 34px, Body: 16px |

#### 4.2 响应式CSS

```css
@media (max-width: 640px) {
  .heading-1 {
    font-size: 28px;
    line-height: 1.3;
  }
  .body-regular {
    font-size: 14px;
  }
}

@media (min-width: 1024px) {
  .heading-1 {
    font-size: 34px;
  }
}
```

## 📐 排版间距

### 1. 字间距 (Letter Spacing)

| 场景 | 字间距 | 说明 |
|------|--------|------|
| 标题 | -0.02em | 紧凑感 |
| 正文 | 0em | 默认 |
| 按钮标签 | 0.03em | 清晰感 |

### 2. 段落间距 (Paragraph Spacing)

```
段落间距 = 行高 × 1.5
- 标题后: 16px
- 正文段落: 12px
- 列表项: 8px
```
