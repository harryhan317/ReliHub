## 📋 模块概述

组件库是所有模块使用的可复用UI组件：
- 🎨 **基础组件**: 按钮、输入框、选择器等
- 📋 **布局组件**: 卡片、列表、网格等
- 🧩 **复合组件**: 导航、标签、徽章等
- 🎭 **状态组件**: 加载、空态、错误状态
- 📱 **响应式**: 支持多种屏幕尺寸

**设计目标**:
- 高度可复用
- 一致的视觉风格
- 完整的交互状态
- Apple Design System 美学

### 1. 核心色彩系统

```
基础色系:
  主色: #007AFF (iOS蓝)
  辅色: #5AC8FA (浅蓝)
  
功能色:
  成功: #34C759 (绿)
  警告: #FF9500 (橙)
  错误: #FF3B30 (红)
  信息: #5AC8FA (浅蓝)
  
中性色:
  纯黑: #000000
  纯白: #FFFFFF
  灰1: #8E8E93 (次级文本)
  灰2: #D1D1D6 (分隔线)
  灰3: #F2F2F7 (背景)
```

### 2. 排版规范

```
H1: 34px / 700w / -0.05em (大标题)
H2: 28px / 700w / -0.02em (页面标题)
H3: 22px / 700w / 0em (分组标题)
H4: 18px / 700w / 0em (小标题)

Body: 16px / 400w / 0.02em (正文)
Label: 12px / 500w / 0.04em (标签)
Caption: 13px / 400w / 0em (辅助文本)

行高: Body 1.5, Label 1.4, Caption 1.5
```

## 🎯 组件详细设计

### 1. 按钮组件

#### 1.1 按钮规格

```
尺寸:
  大 (Large): 48px 高 × 全宽
  标准 (Medium): 44px 高 × 根据内容
  小 (Small): 36px 高 × 根据内容
  迷你 (Mini): 32px 高 × 根据内容

间距:
  水平 Padding: 16px (大), 12px (标准), 8px (小)
  边框圆角: 8px
```

#### 1.2 按钮状态

```
主按钮 (Primary):
  默认: 蓝色 #007AFF
  Hover: 深蓝 #0051D5
  Active: 更深 #003AA3
  禁用: 灰色 #C7C7CC

次按钮 (Secondary):
  默认: 白色底, 灰色文字, 边框
  Hover: 灰色背景
  Active: 更深灰
  禁用: 灰色底, 浅灰文字

文本按钮 (Text):
  默认: 蓝色文字
  Hover: 深蓝背景
  Active: 更深蓝
  禁用: 灰色文字

危险按钮 (Danger):
  默认: 红色 #FF3B30
  Hover: 深红
  Active: 更深红
  禁用: 灰色
```

#### 1.3 Visual Design


## 📖 使用指南

### 按钮使用

```
主操作: 主按钮 (Primary)
次操作: 次按钮 (Secondary)
链接操作: 文本按钮 (Text)
危险操作: 危险按钮 (Danger)
```

### 输入框使用

```
账号输入: 单行文本
密码输入: 隐藏文本
描述输入: 多行文本
日期输入: 日期选择器
```

### 卡片使用

```
内容展示: 基础卡片
可交互: 按钮卡片
图片展示: 图片卡片
统计数据: 数据卡片
```
