## 📋 模块概述

我的模块是用户个人中心，展示用户信息和相关数据：
- 👤 **用户信息**: 头像、昵称、签名、等级
- 📊 **个人数据**: 可可豆、信誉分、贡献度
- 📁 **内容管理**: 我的话题、资源、收藏
- ⚙️ **设置中心**: 个人设置、隐私控制
- 📤 **导出分享**: 个人资料分享和导出

**设计目标**:
- 清晰的信息展示
- 便捷的数据查看
- 快速的功能访问
- Apple Design System 美学

## 🎨 视觉设计


#### 1.2 用户信息卡片

| 属性 | 规格 | 说明 |
|------|------|------|
| 头像 | 80×80px | 圆形 |
| 昵称 | 18px, 700w | 最多 20 字 |
| 签名 | 14px, 400w | 最多 100 字 |
| 数据行 | 16px, 600w | 可可豆、信誉分、等级 |

### 2. 个人资料编辑

#### 2.1 表单字段

| 字段 | 类型 | 验证 | 必填 |
|------|------|------|------|
| 头像 | 图片上传 | 大小 ≤ 5MB | ✗ |
| 昵称 | 文本 | 长度 2-20 | ✓ |
| 签名 | 文本 | 长度 0-100 | ✗ |
| 城市 | 选择 | 预定义列表 | ✗ |
| 职位 | 文本 | 长度 0-50 | ✗ |
| 公开收藏 | 开关 | - | ✗ |

#### 2.2 验证规则

```
昵称:
  ✓ 2-20 字符
  ✓ 不能全是数字
  ✓ 不能包含特殊符号

签名:
  ✓ 0-100 字符
  ✓ 支持 emoji
  ✓ 不能包含链接
```

## 📊 API 映射

### 获取用户信息

```
GET /api/v1/users/me

响应:
{
  "id": "uuid",
  "avatar": "url",
  "nickname": "张三",
  "level": 5,
  "cocoa_beans": 1250,
  "reputation_score": 850,
  "signature": "我是一个硬件工程师 ~",
  "created_at": "2026-01-01T00:00:00Z"
}
```

### 获取我的话题

```
GET /api/v1/users/me/topics?page=1&limit=20

响应:
{
  "topics": [
    {
      "id": "uuid",
      "title": "怎样快速提高PCB设计效率?",
      "views": 128,
      "replies": 24,
      "likes": 32,
      "created_at": "2026-04-05T14:00:00Z",
      "is_featured": true
    }
  ],
  "total": 24
}
```

### 更新用户信息

```
PUT /api/v1/users/me

请求:
{
  "nickname": "新昵称",
  "signature": "新签名",
  "avatar": "avatar_url"
}

响应: 200 OK
```
