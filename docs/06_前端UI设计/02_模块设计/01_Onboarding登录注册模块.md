## 📋 模块概述

Onboarding 登录注册模块是用户进入 ReliHub 平台的第一个交互界面，涵盖：
- 🎯 **游客欢迎页**: 未登录用户首次进入
- 📝 **登录界面**: 存量用户登录
- ✍️ **注册界面**: 新用户注册
- 🔑 **密码重置**: 密码找回流程
- ✅ **注册成功**: 完成欢迎提示

**设计目标**:
- 简洁高效的用户转化流程
- 清晰的视觉层级和交互反馈
- 遵循 Apple Design System 美学
- WCAG AA 无障碍标准

## 🎨 视觉设计

### 1. 设计风格

| 维度 | 标准 | 说明 |
|------|------|------|
| **风格** | 简洁现代 | 高留白、清晰层级 |
| **配色** | Apple 蓝 + 中性灰 | Primary Blue (#007AFF) + White/Gray |
| **圆角** | 8-12px | iOS 风格圆角 |
| **阴影** | 浅色阴影 | 用于浮起效果 |
| **动画** | 300-400ms | 过渡动画时长 |

### 2. 色彩方案

```
主色: #007AFF (Apple Blue)
背景: #FFFFFF (Light) / #000000 (Dark)
文本: #000000 (Light) / #FFFFFF (Dark)
辅助: #8E8E93 (Gray)
```

### 3. 排版规范

| 元素 | 字号 | 字重 | 用途 |
|------|------|------|------|
| 页面标题 | 34px | 700 | "欢迎来到 ReliHub" |
| 正文 | 16px | 400 | 登录表单、描述文字 |
| 按钮文本 | 12px | 500 | "登录"、"注册" |
| 辅助文本 | 14px | 400 | 提示、链接 |

```

│                                 │
│ ┌─────────────────────────────┐ │
│ │   [登录] [注册]              │ │  (CTAs, 按钮区)
│ └─────────────────────────────┘ │
│                                 │
│ [ 以游客身份浏览 ]              │  (次要操作)
│                                 │
```

#### 1.2 组件规格

| 组件 | 规格 | 颜色 | 状态 |
|------|------|------|------|
| **Logo** | 48×48px | 品牌色 | 静态 |
| **标题** | 28px, 700w | #000000 | - |
| **按钮组** | 2×48px高 | Blue/White | Normal/Hover/Press |
| **文本链接** | 14px, 600w | Blue | Normal/Hover/Underline |

#### 1.3 交互设计

- **按钮悬停**: 背景色 -10% 亮度
- **按钮按下**: 背景色 -20% 亮度 + 反馈震动
- **加载状态**: 按钮显示加载指示器

#### 1.4 响应式设计

```
Mobile (<640px):
  - 标题: 24px
  - 按钮宽度: 100%
  - 间距: 16px padding

Desktop (>1024px):
  - 标题: 28px
  - 按钮宽度: 300px
  - 间距: 24px padding
```

### 2. 登录页 (Login Page)

#### 2.1 布局结构

```
┌─────────────────────────────────┐
│ [返回]        登 录       [X]    │  (导航栏)
├─────────────────────────────────┤
│                                 │
│  登录您的账号                   │
│  (H3 22px)                      │
│                                 │
│ ┌─────────────────────────────┐ │
│ │ 📧 邮箱                     │ │  (Email 输入框)
│ │ user@example.com            │ │
│ └─────────────────────────────┘ │
│                                 │  (间距 16px)
│ ┌─────────────────────────────┐ │
│ │ 🔐 密码                     │ │  (Password 输入框)
│ │ ••••••••                    │ │
│ └─────────────────────────────┘ │
│                                 │
│ ┌─────────────────────────────┐ │
│ │   [ 登 录 ]                 │ │  (主操作按钮)
│ └─────────────────────────────┘ │
│                                 │
│ [忘记密码?]  [没有账号? 注册]    │  (辅助链接)
│                                 │
└─────────────────────────────────┘
```

#### 2.2 输入框规格

| 属性 | 值 | 说明 |
|------|-----|------|
| 高度 | 48px | 便于手指触击 |
| 内边距 | 12px | 左右内容间距 |
| 圆角 | 8px | Apple风格 |
| 边框 | 1px solid #E5E5EA | 浅灰色边框 |
| 焦点状态 | 边框 2px #007AFF | 蓝色焦点框 |

#### 2.3 表单验证

```
📧 Email 验证:
  ✓ 非空验证
  ✓ 格式验证 (RFC 5322 简化)
  ✗ 错误提示: 邮箱格式不正确

🔐 Password 验证:
  ✓ 非空验证
  ✓ 长度 >= 8 字符
  ✗ 错误提示: 密码至少8个字符
```

#### 2.4 错误处理

```
登录失败场景:

1. 用户不存在
   提示: "该邮箱未注册"
   操作: [前往注册]

2. 密码错误
   提示: "密码不正确"
   操作: [忘记密码?]

3. 账号已禁用
   提示: "账号已被禁用，请联系客服"
   操作: [联系客服] [返回]

4. 登录频繁
   提示: "登录尝试过多，请5分钟后重试"
   倒计时: 显示剩余时间
```

## ♿ 无障碍设计

### 1. 颜色对比度

- ✅ 文本与背景: ≥4.5:1 (WCAG AA)
- ✅ 按钮与背景: ≥3:1 (UI 组件)

### 2. 触击目标

- ✅ 按钮最小尺寸: 44×44px (iOS标准)
- ✅ 按钮间最小间距: 8px

### 3. 键盘导航

- ✅ Tab 顺序: 顶部→底部
- ✅ 焦点框: 清晰可见
- ✅ Enter 提交表单

### 4. 屏幕阅读器

- ✅ 所有按钮有 aria-label
- ✅ 表单有 label 关联
- ✅ 错误提示正确关联

```html
<!-- 示例 -->
<label for="email">邮箱</label>
<input id="email" type="email" aria-required="true" />
```

## 📊 API 映射

### 登录接口

```
POST /api/v1/auth/login

请求:
{
  "email": "user@example.com",
  "password": "password123"
}

响应:
{
  "access_token": "jwt_token",
  "token_type": "bearer",
  "expires_in": 86400,
  "user": {
    "id": "uuid",
    "email": "user@example.com",
    "username": "alice"
  }
}

错误码:
401 - 用户不存在或密码错误
429 - 登录尝试过多
```

### 注册接口

```
POST /api/v1/auth/register

请求:
{
  "email": "user@example.com",
  "password": "password123",
  "username": "alice"
}

响应:
{
  "id": "uuid",
  "email": "user@example.com",
  "username": "alice",
  "created_at": "2026-04-05T..."
}

错误码:
409 - 邮箱已注册
422 - 数据验证失败
```
