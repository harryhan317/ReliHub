## 📋 模块概述

通知模块提供系统通知和互动提醒：
- 🔔 **系统通知**: 应用更新、维护通知
- 💬 **互动提醒**: 评论、点赞、回复
- 👥 **社区提醒**: 话题更新、关注动态
- 📤 **资源通知**: 资源更新、下载完成
- ⚙️ **通知设置**: 自定义通知偏好

**设计目标**:
- 清晰的通知展示
- 便捷的通知管理
- 丰富的通知类型
- Apple Design System 美学

## 🎨 视觉设计

### 1. 核心色彩

```
主色: #007AFF (通知蓝)
背景: #FFFFFF (浅) / #000000 (深)

未读: #FF3B30 (红 - 强调)
```

### 2. 排版规范

- **标题**: 18px, 700w (H4)
- **正文**: 16px, 400w (Body)

## 📊 API 映射

### 获取通知列表

```
GET /api/v1/notifications?type=all&page=1&limit=20

响应:
{
  "notifications": [
    {
      "id": "uuid",
      "type": "comment",
      "title": "张三评论了你的话题",
      "content": "怎样快速提高PCB...",
      "is_read": false,
      "created_at": "2026-04-05T14:00:00Z"
    }
  ],
  "unread_count": 5,
  "total": 125
}
```

### 标记通知为已读

```
PUT /api/v1/notifications/{id}/read

响应: 200 OK
```

### 更新通知设置

```
PUT /api/v1/notifications/settings

请求:
{
  "enable_push": true,
  "enable_sound": true,
  "enable_vibration": true,
  "comment_notifications": true,
  "like_notifications": true
}

响应: 200 OK
```
