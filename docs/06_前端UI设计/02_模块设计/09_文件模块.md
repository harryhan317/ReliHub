## 📋 模块概述

文件模块管理用户上传和下载的文件：
- 📂 **文件管理**: 上传、删除、分享文件
- 🔍 **文件浏览**: 浏览和搜索文件
- 📋 **文件列表**: 查看文件属性
- 🏷️ **文件分类**: 按类型和标签组织
- 📊 **文件统计**: 查看存储使用情况

**设计目标**:
- 清晰的文件展示
- 便捷的文件操作
- 丰富的文件类型支持
- Apple Design System 美学

### 3. 上传文件

#### 3.1 表单字段

| 字段 | 类型 | 验证 | 必填 |
|------|------|------|------|
| 文件 | 文件上传 | 大小、格式 | ✓ |
| 文件名 | 文本 | 长度 1-255 | ✗ |
| 文件描述 | 文本 | 长度 0-500 | ✗ |
| 分类 | 选择 | 预定义 | ✓ |
| 标签 | 多选 | 自定义 | ✗ |
| 是否公开 | 开关 | - | ✗ |

#### 3.2 验证规则

```
文件:
  ✓ 支持: pdf, doc, xls, ppt, jpg, png, zip, etc
  ✓ 大小: ≤ 100MB
  ✓ 病毒扫描

文件名:
  ✓ 1-255 字符
  ✓ 不能为空
  ✓ 自动使用上传文件名
```

## 📊 API 映射

### 获取文件列表

```
GET /api/v1/files?category=all&sort=recent&page=1

响应:
{
  "files": [
    {
      "id": "uuid",
      "name": "文件名.pdf",
      "size": 2097152,
      "type": "pdf",
      "category": "documents",
      "tags": ["设计", "规范"],
      "downloads": 120,
      "created_at": "2026-04-05T14:00:00Z"
    }
  ],
  "total": 45,
  "usage": {
    "used": 2400000000,
    "total": 10000000000
  }
}
```

### 上传文件

```
POST /api/v1/files/upload

请求: multipart/form-data
{
  "file": <binary>,
  "name": "文件名",
  "category": "documents",
  "description": "文件描述",
  "tags": ["标签1", "标签2"],
  "is_public": false
}

响应: 201 Created
```

### 删除文件

```
DELETE /api/v1/files/{id}

响应: 200 OK
```
