## 📋 模块概述

反馈模块收集用户的bug报告和建议：
- 🐛 **bug报告**: 报告软件问题
- 💡 **功能建议**: 提出产品改进建议
- 📝 **反馈跟踪**: 查看反馈状态
- 💬 **社区讨论**: 与其他用户讨论
- 📊 **反馈统计**: 查看反馈数据

**设计目标**:
- 简洁的反馈表单
- 清晰的状态反馈
- 便捷的跟踪管理
- Apple Design System 美学

### 3. 提交反馈

#### 3.1 表单字段

| 字段 | 类型 | 验证 | 必填 |
|------|------|------|------|
| 反馈类型 | 选择 | bug/建议/其他 | ✓ |
| 标题 | 文本 | 长度 5-100 | ✓ |
| 描述 | 富文本 | 长度 20-5000 | ✓ |
| 截图/视频 | 文件上传 | 支持格式 | ✗ |
| 设备信息 | 自动 | 自动检测 | - |
| 操作系统 | 文本 | 自动获取 | - |
| 应用版本 | 文本 | 自动获取 | - |
| 优先级 | 选择 | 低/中/高 | ✓ |

#### 3.2 验证规则

```
标题:
  ✓ 5-100 字符
  ✓ 简明扼要
  ✓ 避免重复

描述:
  ✓ 20-5000 字符
  ✓ 详细说明问题
  ✓ 包含复现步骤

截图:
  ✓ 支持: jpg, png
  ✓ 大小: ≤ 10MB/张
  ✓ 最多 5 张
```

## 📊 API 映射

### 获取反馈列表

```
GET /api/v1/feedback?type=bug&status=open&page=1

响应:
{
  "feedback": [
    {
      "id": "uuid",
      "title": "登录页面闪退",
      "type": "bug",
      "status": "processing",
      "priority": "high",
      "author": "张三",
      "views": 500,
      "replies": 8,
      "upvotes": 45,
      "created_at": "2026-04-05T14:00:00Z"
    }
  ],
  "total": 245
}
```

### 提交反馈

```
POST /api/v1/feedback

请求:
{
  "title": "登录页面闪退",
  "description": "在iPhone 12上会闪退...",
  "type": "bug",
  "priority": "high",
  "device_info": {
    "os": "iOS",
    "version": "14.5",
    "app_version": "v1.0.2"
  },
  "screenshots": ["url1", "url2"]
}

响应: 201 Created
```
