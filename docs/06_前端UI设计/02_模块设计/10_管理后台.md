## 📋 模块概述

管理后台为系统管理员提供内容和用户管理：
- 👥 **用户管理**: 用户列表、审核、封禁
- 📝 **内容审核**: 话题、资源审核
- 📊 **数据统计**: 平台数据、趋势分析
- ⚙️ **系统设置**: 参数配置、权限管理
- 🚨 **举报管理**: 举报处理、反馈跟进

**设计目标**:
- 高效的管理界面
- 清晰的数据展示
- 便捷的操作流程
- Apple Design System 美学

## 🎨 视觉设计

### 1. 核心色彩

```
主色: #6B21A8 (管理紫)
背景: #F9FAFB (浅) / #0F172A (深)
文本: #111827 (浅) / #F9FAFB (深)
成功: #10B981 (绿)
警告: #FF9500 (橙)
错误: #EF4444 (红)
信息: #3B82F6 (蓝)
```

### 2. 排版规范

- **标题**: 24px, 700w (H2)
- **副标题**: 18px, 700w (H3)
- **正文**: 14px, 400w (Body)
- **数据**: 16px, 700w (数字)

## 📊 API 映射

### 获取用户列表

```
GET /api/v1/admin/users?page=1&limit=20&status=all

响应:
{
  "users": [
    {
      "id": "uuid",
      "nickname": "张三",
      "email": "zhangsan@email.com",
      "level": 5,
      "contribution": 0.72,
      "status": "active",
      "registered_at": "2026-01-15T00:00:00Z",
      "last_active": "2026-04-05T12:00:00Z"
    }
  ],
  "total": 1250
}
```

### 内容审核

```
PUT /api/v1/admin/content/{id}/review

请求:
{
  "status": "approved",
  "notes": "已通过审核"
}

响应: 200 OK
```
