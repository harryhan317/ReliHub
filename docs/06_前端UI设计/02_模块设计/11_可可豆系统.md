## 📋 模块概述

可可豆系统是平台的虚拟奖励机制：
- 💰 **可可豆管理**: 查看、获取、使用可可豆
- 🎁 **兑换商城**: 用可可豆兑换权益
- 📊 **统计数据**: 可可豆收入和支出
- 🏆 **成就系统**: 解锁徽章和等级
- 💳 **钱包管理**: 可可豆账户管理

**设计目标**:
- 激励用户参与
- 明确的奖励规则
- 便捷的兑换流程
- Apple Design System 美学

## 📊 API 映射

### 获取可可豆信息

```
GET /api/v1/cocoa-beans/balance

响应:
{
  "balance": 1250,
  "level": 5,
  "experience": 2250,
  "next_level_exp": 2500,
  "monthly_income": 450,
  "monthly_expense": 100
}
```

### 兑换商品

```
POST /api/v1/cocoa-beans/exchange

请求:
{
  "product_id": "premium_1month",
  "quantity": 1
}

响应: 201 Created
```

### 获取成就

```
GET /api/v1/cocoa-beans/achievements

响应:
{
  "achievements": [
    {
      "id": "uuid",
      "name": "新手用户",
      "icon": "url",
      "unlocked": true,
      "unlocked_at": "2026-01-15T00:00:00Z"
    }
  ]
}
```
