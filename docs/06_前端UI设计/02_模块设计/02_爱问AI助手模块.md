## 📋 模块概述

爱问 AI 助手模块是 ReliHub 的核心功能，提供：
- 🤖 **AI 对话界面**: 与 AI 助手实时对话
- 📎 **附件上传**: 支持文件、图片、文档上传
- 💬 **会话管理**: 多个对话会话管理
- 📊 **结果展示**: 富文本、表格、代码展示
- 🔖 **收藏标记**: 保存重要对话

**设计目标**:
- 流畅的对话体验
- 丰富的信息展示
- 清晰的附件处理
- Apple Design System 美学

## 🎨 视觉设计

### 1. 设计风格

| 维度 | 标准 | 说明 |
|------|------|------|
| **颜色** | AI Purple (#A78BFA) | AI 相关元素主题色 |
| **间距** | 16px | 标准间距 |
| **圆角** | 8-12px | Apple 风格 |
| **排版** | Apple System Font | 系统字体 |

### 2. 核心色彩

```
主色: #A78BFA (AI Purple)
背景: #FFFFFF (浅) / #000000 (深)
文本: #000000 (浅) / #FFFFFF (深)
消息气泡 - 用户: #007AFF (蓝)
消息气泡 - AI: #F2F2F7 (浅灰) / #2C2C2E (深灰)
```

## 📊 API 映射

### 获取会话列表

```
GET /api/v1/ai/sessions

响应:
{
  "sessions": [
    {
      "id": "uuid",
      "title": "关于可靠性设计",
      "preview": "可靠性设计的第一步...",
      "created_at": "2026-04-05T14:00:00Z",
      "updated_at": "2026-04-05T15:30:00Z"
    }
  ]
}
```

### 发送消息

```
POST /api/v1/ai/sessions/{session_id}/messages

请求:
{
  "content": "如何设计高可靠性电路?",
  "attachments": ["file_id_1", "file_id_2"]
}

响应:
{
  "id": "uuid",
  "role": "assistant",
  "content": "高可靠性电路设计包括...",
  "created_at": "2026-04-05T15:31:00Z"
}
```
