## 📋 模块概述

搜索模块提供全局搜索和内容发现：
- 🔍 **全局搜索**: 跨模块搜索
- 📌 **智能建议**: 搜索历史和推荐
- 🏷️ **分类过滤**: 按类型筛选结果
- ⚡ **热搜榜单**: 实时热点话题
- 💾 **搜索历史**: 保存搜索记录

**设计目标**:
- 快速的搜索体验
- 丰富的结果展示
- 智能的建议提示
- Apple Design System 美学

```

响应:
{
  "suggestions": [
    {
      "text": "PCB设计",
      "count": 2340,
      "type": "topic"
    },
    {
      "text": "PCB制造",
      "count": 1850,
      "type": "keyword"

}
```

```
GET /api/v1/search?q=PCB&type=all&page=1&limit=20
```
