## 📋 模块概述

资源模块支持技术资源的上传、发现和下载：
- 📤 **资源上传**: 支持文件、文档、代码发布
- 🔍 **资源浏览**: 分类浏览和搜索资源
- 📊 **资源详情**: 预览、下载、评价
- ⭐ **收藏管理**: 保存喜欢的资源
- 📈 **排序筛选**: 按热度、时间、评分排序

**设计目标**:
- 清晰的内容展示
- 便捷的资源发现
- 流畅的下载体验
- Apple Design System 美学

### 3. 发布资源

#### 3.1 表单字段

| 字段 | 类型 | 验证 | 必填 |
|------|------|------|------|
| 资源名称 | 文本 | 长度 3-100 | ✓ |
| 资源描述 | 富文本 | 长度 10-5000 | ✓ |
| 文件上传 | 文件 | 支持类型 + 大小 | ✓ |
| 分类 | 选择 | 单选 | ✓ |
| 标签 | 多选 | 预定义或自定义 | ✗ |
| 是否开源 | 开关 | - | ✗ |

#### 3.2 验证规则

```
资源名称:
  ✓ 3-100 字符
  ✓ 不能全是数字
  ✓ 不能包含特殊字符

描述:
  ✓ 10-5000 字符
  ✓ 支持格式化文本

文件:
  ✓ 支持: pdf, doc, ppt, xls, zip, code
  ✓ 大小: ≤ 100MB
  ✓ 病毒扫描
```

## 📊 API 映射

### 获取资源列表

```
GET /api/v1/resources?category=all&sort=hot&page=1

响应:
{
  "resources": [
    {
      "id": "uuid",
      "title": "可靠性设计指南",
      "size": "2.1MB",
      "author": "张三",
      "rating": 4.8,
      "downloads": 2300,
      "created_at": "2026-04-05T14:00:00Z"
    }
  ],
  "total": 1250
}
```

### 下载资源

```
GET /api/v1/resources/{id}/download

响应: 文件二进制数据 (Content-Disposition: attachment)
```
