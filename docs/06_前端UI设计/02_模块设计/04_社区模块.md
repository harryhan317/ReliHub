## 📋 模块概述

社区模块支持技术讨论和经验分享：
- 💬 **话题讨论**: 发起和参与技术话题
- 👥 **社区互动**: 回复、点赞、收藏
- 🏆 **社区排名**: 用户贡献排行
- 🔔 **话题通知**: 跟进讨论动态
- 📌 **精华管理**: 优质内容推荐

**设计目标**:
- 营造活跃讨论氛围
- 简洁的内容展示
- 丰富的互动反馈
- Apple Design System 美学


### 3. 发起话题

#### 3.1 表单字段

| 字段 | 类型 | 验证 | 必填 |
|------|------|------|------|
| 话题标题 | 文本 | 长度 3-100 | ✓ |
| 话题内容 | 富文本 | 长度 10-10000 | ✓ |
| 话题分类 | 选择 | 单选 | ✓ |
| 标签 | 多选 | 预定义 3-5 个 | ✗ |
| 是否匿名 | 开关 | - | ✗ |

#### 3.2 验证规则

```
话题标题:
  ✓ 3-100 字符
  ✓ 避免重复
  ✓ 不能全是符号

内容:
  ✓ 10-10000 字符
  ✓ 支持代码块
  ✓ 支持图片 (最多 5 张)
  ✓ 自动检测链接

标签:
  ✓ 3-5 个标签
  ✓ 预定义列表
```

## 📊 API 映射

### 获取话题列表

```
GET /api/v1/community/topics?type=hot&page=1

响应:
{
  "topics": [
    {
      "id": "uuid",
      "title": "怎样快速提高PCB设计效率?",
      "author": "张三",
      "content": "...",
      "views": 128,
      "replies": 24,
      "likes": 32,
      "created_at": "2026-04-05T14:00:00Z"
    }
  ],
  "total": 450
}
```

### 发起话题

```
POST /api/v1/community/topics

请求:
{
  "title": "...",
  "content": "...",
  "category": "PCB",
  "tags": ["PCB", "设计"],
  "anonymous": false
}

响应: 201 Created
```
