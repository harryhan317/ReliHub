### 3. Sprint C 技术方案 (20 分钟)

#### 3.1 技术架构
- LLM Provider 架构
- 支付网关架构
- 敏感词过滤架构

#### 3.2 关键技术选型
- **LLM**: OpenAI + Claude + Moonshot
- **支付**: 微信支付（主要）+ 支付宝（备选）
- **敏感词**: AC 自动机算法

#### 3.3 数据库变更
- 新增 `llm_providers` 表
- 新增 `payment_orders` 表
- 新增 `payment_transactions` 表

- 修改 `ai_messages` 表

#### 3.4 API 变更
- 修改 AI 对话 API（支持流式响应）
- 新增支付订单 API
- 新增支付回调 API
