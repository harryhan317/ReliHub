## 一、模块目的与范围边界

### 1.1 文档目的

本文档定义 ReliHub Phase 3 阶段管理后台的扩展功能，与 [PRD_MVP_管理后台.md](../03_MVP_PRD/PRD_MVP_管理后台.md) 配合使用。Phase 3 在 MVP/Phase 2 功能基础上新增专家咨询抽佣配置、广告管理、广告数据统计三个功能模块。

### 1.2 涵盖范围（对应功能清单 M5-F054~F056, M5-F024）

- **RAG知识库管理**：上传/更新专业知识库文档，配置切片策略与向量检索参数（M5-F024）
- **专家咨询抽佣比例配置**：配置平台从专家一对一咨询收益中抽取的佣金比例
- **广告管理**：广告内容的上下架管理
- **广告数据统计**：广告展示量/点击量等数据查看

### 1.3 与 MVP/Phase 2 管理后台的关系

本文档是 [PRD_MVP_管理后台.md](../03_MVP_PRD/PRD_MVP_管理后台.md) 和 [PRD_Phase2_管理后台.md](../04_Phase2_PRD/PRD_Phase2_管理后台.md) 的补充，不重复定义前期已有的功能规则。Phase 3 新增功能在本文档中详细定义。

## 三、专家咨询抽佣比例配置（M5-F054）

### 3.1 功能概述

超级管理员可配置平台从专家一对一咨询收益中抽取的佣金比例。

### 3.2 抽佣配置规则

| 配置项 | 说明 | 默认值 |
|-------|------|--------|
| 抽佣比例 | 平台从每笔咨询订单中抽取的佣金比例 | 20% |
| 可配置范围 | 允许配置的抽佣比例区间 | 10%~30% |
| 生效时间 | 配置变更后何时生效（立即/次日/次月） | 次日0点 |

### 3.3 抽佣统计

| 字段 | 说明 |
|-----|------|
| 今日咨询订单数 | 当日成交的一对一咨询订单总数 |
| 今日咨询总流水 | 当日咨询订单的总金额 |
| 今日平台佣金收入 | 当日平台抽取的佣金总额 |
| 本月累计佣金收入 | 当月累计佣金收入 |

### 3.4 配置审计

- 抽佣比例变更记录永久可查（含操作人/操作时间/变更前后值）
- 调整抽佣比例前需二次确认

## 四、广告管理（M5-F055）

### 4.1 功能概述

超级管理员可在管理后台上传、编辑、删除广告内容，管理广告的展示与投放。

### 4.2 广告列表

**列表字段**：

| 字段 | 说明 |
|-----|------|
| 广告ID | 系统生成的唯一标识 |
| 广告名称 | 广告内容标题 |
| 广告类型 | Banner广告 / 插屏广告 / 贴片广告 |
| 投放位置 | 首页 / AI问答页 / 资源详情页 等 |
| 投放状态 | 上线 / 下线 / 草稿 |
| 点击率 | CTR（点击量/展示量） |
| 创建时间 | 广告创建时间 |
| 操作 | 编辑 / 下线 / 删除 |

**筛选条件**：

| 字段 | 筛选器类型 |
|-----|-----------|
| 投放状态 | 下拉单选 |
| 广告类型 | 下拉单选 |
| 投放位置 | 下拉多选 |
| 创建时间 | 起止日期选择器 |

### 4.3 广告创建/编辑

| 字段 | 说明 | 必填 |
|-----|------|------|
| 广告名称 | 广告内容标题 | ✅ |
| 广告类型 | Banner/插屏/贴片 | ✅ |
| 投放位置 | 选择展示页面 | ✅ |
| 广告素材 | 图片/视频上传 | ✅ |
| 跳转链接 | 点击后跳转URL | ✅ |
| 展示时间段 | 广告投放起止日期 | ✅ |
| 每日展示上限 | 防止广告曝光过度 | ❌ |

### 4.4 广告审核规则

- 广告内容需经过审核后方可上线（审核标准同社区内容）
- 涉及医疗/金融类广告需额外资质文件

## 五、广告数据统计（M5-F056）

### 5.1 功能概述

运营负责人可查看广告的展示量、点击量、点击率等核心数据。

### 5.2 数据概览

| 指标 | 说明 |
|-----|------|
| 今日展示量 | 当日广告总展示次数 |
| 今日点击量 | 当日广告总点击次数 |
| 今日点击率 | 点击量/展示量（CTR） |
| 今日广告收入 | 预估广告收入（按CPM/CPC结算） |
| 环比变化 | 与昨日同期数据对比 |

### 5.3 分广告明细

| 字段 | 说明 |
|-----|------|
| 广告名称 | 广告内容标题 |
| 展示量 | 累计展示次数 |
| 点击量 | 累计点击次数 |
| CTR | 点击率 |
| 转化数 | 广告引导的用户行为数（如下载/注册） |
| 广告收入 | 该广告累计收入 |

### 5.4 数据导出

支持按时间范围、广告类型、投放位置等条件导出数据报表（CSV格式）。

## 六、RAG知识库管理（M5-F024）

### 6.1 功能概述

超级管理员可在管理后台上传、更新、删除可靠性领域专业文档，配置 RAG 切片策略与向量检索参数，构建 AI 助手的专业知识库。

### 6.2 文档管理

**支持的文档格式**：PDF、Word (.docx)、Txt、Markdown、Excel (.xlsx/.csv)。

| 字段 | 说明 |
|-----|------|
| 文档名称 | 上传的文档名称 |
| 文档类型 | 可靠性设计标准 / 器件手册 / 测试方法 / 失效分析 / 质量管控 / 工艺规范 |
| 上传时间 | 文档上传时间 |
| 文档状态 | 待解析 / 解析中 / 已入库 / 解析失败 |
| 操作 | 预览 / 重新解析 / 删除 |

**上传规则**：
- 单文件最大 50MB
- 文档经过 AI 安全预审（敏感内容检测）后方可入库
- 解析失败时管理员可下载查看错误详情并重新上传

### 6.3 切片策略配置

| 参数名称 | 可调范围 | 默认值 | 说明 |
|---------|---------|-------|------|
| 文档切片大小 | 100~2000字符 | 500字符 | 每段文本的字符数 |
| 切片重叠量 | 0~200字符 | 50字符 | 相邻切片之间的重叠字符数 |
| 向量检索召回数量（Top-K） | 1~20 | 5 | 检索时返回的最相似片段数量 |
| RAG相关性阈值 | 0.10~1.00 | 0.7 | 片段与问题的最低相关度阈值 |

### 6.4 知识库内容范围

- 可靠性设计标准与规范
- 器件手册与应用笔记
- 测试方法与实验标准
- 失效分析案例与方法
- 质量管控流程与工具
- 工艺可靠性最佳实践

### 6.5 向量检索配置

超级管理员可配置以下向量检索参数：

| 参数 | 说明 |
|-----|------|
| Embedding 模型 | 选择使用的向量化模型（如 text-embedding-ada-002） |
| 向量维度 | 默认为 1536（Ada v2） |
| 索引类型 | HNSW / IVF-PQ 等 |
| 检索模式 | 精确检索 / 近似最近邻（ANN） |
