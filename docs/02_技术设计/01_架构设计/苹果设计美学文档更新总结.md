**关键变更**:

**§8.0 设计哲学**:
```markdown
**ReliHub 采用苹果设计美学 (Apple Design Philosophy) 作为产品交互界面的设计指导方针。**

核心原则:
- 简洁至上：少即是多，去除一切不必要的元素
- 内容为王：让内容成为焦点，界面为内容服务
- 直观易用：无需说明书，用户凭直觉就能使用
- 一致性：整个产品保持统一的设计语言
- 人性化关怀：尊重用户，关怀用户，让技术有温度
```

**§8.2.1 视觉规范 (苹果美学)**:
- 配色方案：深空灰 (#1D1D1F) / 苹果蓝 (#0071E3)
- 字体系统：优先使用系统字体 (SF Pro/PingFang SC)
- 图标风格：简洁线性图标 (Lucide React)
- 间距规范：8px 栅格系统，留白充足
- 圆角规范：按钮 8px / 卡片 12px / 模态框 16px
- 阴影规范：柔和自然，避免生硬黑影
- 留白艺术：充足留白突出内容

**§8.2.2 动效设计 (苹果风格)**:
- 页面切换：200-300ms, cubic-bezier(0.4, 0.0, 0.2, 1)
- 弹窗显示：200ms, 轻微缩放
- 按钮点击：100ms, scale 0.96
- 加载动画：简洁圆形 spinner
- 成功反馈：轻微弹跳动画

## 🎨 核心设计规范速查

### 色彩系统

```css
/* 主色 - 苹果深空灰 */
--color-primary: #1D1D1F;        /* 深空灰 */
--color-accent: #0071E3;         /* 苹果蓝 - 强调色 */
--color-success: #34C759;        /* 苹果绿 */
--color-warning: #FF9500;        /* 苹果黄 */
--color-error: #FF3B30;          /* 苹果红 */
```

**使用原则**: 60-30-10 法则

---

### 字体系统

```css
--font-family: -apple-system, BlinkMacSystemFont, "SF Pro Text";
--text-base: 17px;  /* iOS 默认正文字号 */
```

---

### 间距系统 (8px 网格)

```css
--space-4: 16px;   /* 标准间距 */
--space-6: 24px;   /* 大间距 */
--space-12: 48px;  /* 模块间距 */
```

**原则**: 宁大勿小，充足留白

---

### 交互动效

```css
/* 按钮点击反馈 */
.button:active {
  transform: scale(0.96);
  transition: transform 100ms ease-standard;
}

/* 模态框弹出 */
.modal-enter {
  opacity: 0;
  transform: scale(0.95) translateY(10px);
  transition: all 200ms ease-out-slow;
}
```

## 🛠️ 技术栈更新

### 新增依赖

```json
{
  "framer-motion": "^10.x",    // 苹果式流畅动画
  "lucide-react": "^0.x"       // 简洁线条图标
}
```

### 使用示例

**Framer Motion**:
```tsx
import { motion } from 'framer-motion';

<motion.button
  whileTap={{ scale: 0.96 }}
  transition={{ duration: 0.1 }}
>
  点击我
</motion.button>
```

**Lucide React**:
```tsx
import { Home } from 'lucide-react';

<Home size={24} strokeWidth={2} />
```
