# 集成方案 - 文件预览方案 (集成_文件预览方案)

## 1. 预览策略
- **PDF 预览**: 前端集成 `pdf.js` 直接渲染。
- **Office 预览**: 后端 Worker 使用 LibreOffice 执行转码并返回 PDF 流。
- **图片预览**: 前端直接渲染轻量级缩略图。

## 2. 预览限制
- 对齐 PRD：游客仅能预览资源的前 N 页（由后端截断 PDF 流或返回特定封面）。
- 预览界面强制覆盖水印。

### 2.3 非标格式处理 (TIF/TIFF)
- **转码要求**: TIF 格式必须在上传后由异步任务强制转为 JPG/PNG 格式。
- **存储**: 转码后的预览图存储在 OSS `previews/` 目录下。
- **SLA**: TIF 转码属于后台任务，不计入 5s 预览响应 SLA。
