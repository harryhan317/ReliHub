# 集成方案 - 微信登录方案 (集成_微信登录方案)

## 1. 场景覆盖
- 微信 H5 (微信浏览器内)。
- 微信小程序 (Phase 2)。
- 微信扫码登录 (Web 端)。

## 2. 交互流程 (H5 示例)
1. 前端跳转至微信授权 URL。
2. 微信回调后端接口返回 `code`。
3. 后端调用微信接口通过 `code` 换取 `access_token` 与 `openid / unionid`。
4. 后端查找 `users` 表，如果存在则签发 JWT；不存在则创建新用户。
