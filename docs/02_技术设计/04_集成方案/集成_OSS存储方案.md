# 集成方案 - OSS 存储方案 (集成_OSS存储方案)

## 1. 存储策略
- **私有桶 (Private Bucket)**: 存储原始资源文件、用户附件。
- **公共桶 (Public Bucket)**: 存储头像、社区话题/帖子插图、静态前端资源。

## 2. 安全访问
- 禁止任何匿名访问私有资源。
- 服务端生成 STS (Security Token Service) 临时凭证给前端直传。
- 下载时后端根据权限校验结果，返回带有 `Signature` 的短效临时 URL (有效期 5min)。

## 3. 防盗链与地区优化
- 开启 OSS 白名单 Referer 检查。
- MVP 阶段开启国内 CDN 加速。
