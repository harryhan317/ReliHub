 ## 演进原则
 1. **对齐优先**：技术设计必须 100% 覆盖 PRD 定义的业务逻辑。
 2. **解耦原则**：前端 React 与后端 FastAPI 通过 RESTful API 解耦。
 3. **安全内置**：所有接口需验证 PII 脱敏与权限鉴权（尤其是管理员接口）。
